# Omnitalk FLUTTER SDK

[옴니톡](omnitalk.io)은 webRTC 표준 기술을 이용하여 app에서 쉽고 간편하게 실시간 통신을 구현할 수 있는 SDK입니다.<br/>


<br/><br/>

## Feature Overview

| feature          | implemented | ios | android |
| ---------------- | :---------: | :-: | :-----: |
| Audio Call       |     ✔️      | ✔️  |   ✔️    |
| Video Call       |     ✔️      | ✔️  |   ✔️    |
| Sip Call         |     ✔️      |  ✔️ |   ✔️    |
| Chatting         |     ✔️      | ✔️  |   ✔️    |
| Audio Conference |     ✔️      | ✔️  |   ✔️    |
| Video Conference |     ✔️      | ✔️  |   ✔️    |
| AudioMute        |     ✔️      | ✔️  |   ✔️    |
| VideoMute        |     ✔️      | ✔️  |   ✔️    |
| Video Switch     |     ✔️      |  ✔️ |   ✔️    |
